プライバシーポリシー

BEYOND Holdings 株式会社 (以下「当社」) は、当社が提供するWebサイト「BEYOND PLAYBOOK」(https://playbook.beyond-holdings.co.jp/) におけるお客様の個人情報の取扱いについて、以下のとおり定めます。本ポリシーは、個人情報保護法その他関連法令を遵守し、お客様の個人情報を適切に保護することを目的としています。

1. 取得する個人情報

当社は、お問い合わせフォーム・お問合せフォーム・診断ツール等を通じて、以下の情報を取得することがあります。

  • お名前(姓名)
  • 会社名・業種・年商規模
  • ご連絡先(電話番号 / メールアドレス / LINE ID 等)
  • ご相談内容・緊急度
  • 診断ツール回答内容
  • アクセスログ情報(IPアドレス・ブラウザ・参照元URL・閲覧ページ等)

2. 利用目的

取得した個人情報は、以下の目的に限り利用します。

  • お問い合わせ・ヒアリング申込への対応
  • 当社サービスのご案内・ご提案
  • お客様への重要なお知らせの送信
  • 当社サービスの改善・新サービス開発のための分析
  • 法令に基づく対応

3. 第三者への提供

当社は、以下の場合を除き、お客様の同意なく個人情報を第三者に提供することはありません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合
  • 当社が業務委託先(クラウドサービス事業者等) に対して、利用目的の達成に必要な範囲で開示する場合(委託先には適切な監督を行います)

4. BEYOND Holdings グループ会社内での共有

お客様が当社のサービスを通じてご相談された課題の解決にあたり、当社が単独で対応するよりも、 ★BEYOND Holdings グループの他社が持つ知見・リソースを活用することが最適と判断される場合★ があります。その際は、お客様の解決に必要な最小限の範囲で、グループ会社間で情報を共有することがあります。共有先は以下のグループ会社に限定します。

  • 株式会社デンコー (電力機器事業)
  • 株式会社SAFARI (EC・メディア事業)
  • 株式会社HANABI (美容サービス事業)
  • 株式会社K2 (飲食事業)
  • 株式会社ライフプロ (不動産事業)
  • 株式会社M7Logi (アパレル・イベント事業)
  • 株式会社184 (リラクゼーション事業)
  • 株式会社DNK (アプリ運営事業)

グループ会社内であっても、本プライバシーポリシーと同等の保護基準を適用します。

5. 業務委託先・利用ツール

当社は、サービス運営にあたり以下の外部サービスを利用することがあります。これらのサービスを通じて、お客様の情報が当該事業者のサーバーに保管・処理されることがあります。

  • Cloudflare, Inc.: Webサイトのホスティング・配信・アクセス解析
  • LINE WORKS (ワークスモバイルジャパン株式会社): お問い合わせ通知・社内コミュニケーション
  • Box, Inc.: ドキュメント保管・ファイル共有
  • GitHub, Inc.: ソースコードのバージョン管理
  • Anthropic, PBC: AI処理 (Claude API)
  • Google LLC: Google Analytics 等のアクセス解析・各種クラウドサービス
  • その他: サービス運営上必要な範囲で、適切に管理された業務委託先を利用することがあります

各事業者のプライバシーポリシーについては、各社の公式サイトをご確認ください。

6. Cookie・アクセス解析

当サイトでは、サービス向上のためCookieを使用することがあります。Cookieによって個人を特定する情報は取得しません。また、Cloudflare Web AnalyticsおよびGoogle Analytics等のアクセス解析ツールを利用する場合、これらのツールはCookieを使用してトラフィックデータを収集することがあります。お客様はブラウザの設定によりCookieを無効化することができます。

7. 個人情報の管理

当社は、お客様の個人情報について、不正アクセス・紛失・改ざん・漏洩等を防止するため、合理的な安全管理措置を講じます。情報は必要な期間のみ保管し、不要となった情報は適切に削除・廃棄します。

8. 開示・訂正・削除の請求

お客様ご本人から、ご自身の個人情報の開示・訂正・削除・利用停止の請求があった場合、本人確認の上、合理的な範囲で速やかに対応いたします。ご請求は下記窓口までご連絡ください。

9. プライバシーポリシーの改定

本ポリシーは、法令の変更・サービス内容の変更等に応じて、予告なく改定することがあります。改定後のポリシーは、本ページに掲載した時点から効力を生じます。
