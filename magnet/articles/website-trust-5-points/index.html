『ちゃんとした会社』に
見せる5つのポイント

取引前に相手の会社名を検索する――いまや当たり前の動作です。そして検索の入口は、PCからスマホへ、さらにAIの回答へと移りました。Googleはモバイル版を主に見て評価し（モバイルファーストインデックス）、表示の速さや画面の崩れ（Core Web Vitals）は検索順位にも直結します。つまり、HPの第一印象が会社そのものの信頼として読まれる時代。見た目を整えるのは、見栄の問題ではなく受注の問題です。

1.いま起きている事実 ── 入口が「スマホとAI」に変わった

かつて会社の信頼は、紹介や名刺、立派な事務所で測られました。いまは違います。見込み客はまず手元のスマホで社名を検索し、出てきたHPの第一画面で「この会社、大丈夫そうか」を数秒で判断します。Googleが評価の主軸をモバイル版に置き、表示の速さや崩れのなさ（Core Web Vitals）を順位の要素にしているのも、利用者の体験がそれだけ重視されているからです。

さらに最近は、AIに「◯◯ができる会社は?」と尋ねる人も増えました。AIは整理され信頼できる情報を引用しやすく、逆に古く崩れたページは拾われにくい。検索の入口が増えた分、第一印象で振るい落とされる場面も増えた。技術や実績があっても、入口の見え方で会う前に脱落するのは、あまりにもったいない話です。

2.これが意味すること ── 「立派さ」より「不安をなくす」

誤解されがちですが、信頼される会社に見せることは「お金をかけて豪華にする」こととは違います。相手が見ているのは華やかさではなく、『この会社は実在し、ちゃんと連絡が取れて、後でトラブルにならないか』という不安の有無。第一印象とは、その不安を一つずつ消せているかどうかなのです。

だから、デザインの巧拙より先に潰すべきは『不安の素』。会社の実体が見えない、スマホで読めない、情報が何年も古い――こうした要素は、内容を読む前に相手を帰らせます。逆に言えば、不安を消す要素を押さえるだけで、特別な予算がなくても『ちゃんとした会社』に見せられます。

3.自社ならどう動くか ── 信頼を作る5つのポイント

①スマホで『速く・崩れず』開けるか確認する
アクセスの多くはスマホ。指で読める文字サイズ、はみ出さない画像、待たせない表示。重い・崩れる時点で内容は読まれません。こんな現場：PCでしか自社サイトを確認していない
②『誰が・どこで』やっているかを顔と住所で示す
代表の顔写真、所在地、電話、設立年。実体が見える情報は最強の信頼材料。実在性が伝わるだけで不安は大きく減ります。こんな現場：会社概要が最小限で人の気配がない
③写真は『現場の実物』を使う
フリー素材の握手写真より、自社の仕事・人・現場の一枚。多少素朴でも本物の写真のほうが、嘘がなく信用できると伝わります。こんな現場：トップが借り物のイメージ画像だけ
④『お客様の声・実績』で第三者に語らせる
自社の自慢は割引かれて読まれる。取引先の声や事例、取引実績は、自分で言うより何倍も効く信頼の証明になります。こんな現場：強みを自社の主張だけで押している
⑤情報の『鮮度』を保つ
更新が数年止まったお知らせは「動いていない会社」の合図。日付の古さは即マイナス。直近の更新が一つあるだけで印象は変わります。こんな現場：最終更新が何年も前のまま
第一印象は、会う前の面接。
不安を消した会社だけが、次のページに進んでもらえます。

4.最初の一手は「自分のスマホで開くこと」

いきなり全面リニューアルは要りません。まずやるべきは、自分のスマホで自社サイトをお客様として開いてみること。遅い、崩れる、誰がやっているか分からない、写真が借り物、更新が止まっている――気になった点が、そのまま相手を帰らせている『不安の素』です。一つ消すごとに、第一印象は確実に上がります。

大切なのは、作り手の自己満足で整えないこと。初めて訪れた相手の不安から逆算する。MAGNETは、この『選ばれる見え方』を、現場で続く形に落とし込みます。
